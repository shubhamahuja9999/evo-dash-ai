## 🚀 **How to Use (Super Simple)**

### **Step 1: Open the Dashboard**
1. Start your app: `npm run dev`
2. Click "Google Ads" in the sidebar
3. See your complete automation dashboard

### **Step 2: Try the One-Click Actions**
```
Click any of these buttons for instant results:
• "Optimize Bids" → Improves keyword performance
• "Find Negative Keywords" → Stops wasted spend
• "Run Daily Optimization" → Complete account tune-up
• "Check Account Billing" → Ensures payments are working
```

### **Step 3: Use the AI Chat**
```
Click "AI Assistant" and type commands like:
• "Stop campaigns costing over $50 per lead"
• "Show me my best performing keywords"
• "Create a new campaign for software engineers"
• "Why did my costs go up this week?"
```

### **Step 4: Set Up Automation**
```
Click "Create Automation Task" and choose:
• Task Type: Negative Keyword Detection
• Schedule: Daily at 6:00 AM
• Click "Create Task" → Done!
```
